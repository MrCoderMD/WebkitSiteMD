Like lists, Table is also a way to display data in a mannered way.

But the key difference is, that while lists sho data in bullets or numberings, tables show data in form of rows and columns.

HTML gives us a lot of flexibility to make different types of tables.

Initializing table

To start making a table, we use <table> tag.

<table></table>

Making table rows

To make table rows we use <tr> tag

<table><tr></tr></table>

<tr> creates a row in the table which will contain cells in it.

creating table header and cells

A table header is used to define the head of the column. It is just like a normal table cell but with bold text inside of it.

A <th> tag is used to crete a table header while a <td> tag is used to create a table cell.

<table>
<tr>
<th></th>
<th></th>
</tr>
<tr>
<td></td>
<td></td>
</tr>
</table>

borders, cellspacing and cellpadding

To create a border, a border attribute is given with the table tag.

<table border='5px'></table>

The value for the border attribute should be a length.

Both cellpadding and cellspacing attributes are given to the table tag. Cellspacing defines the space between the cells in all directions while cellpadding defines space between cell content and cell border.

<table border="5px" cellspacing="10px" cellpadding="10px">
<tr>
<th>Head</th>
</tr>
<tr>
<td>cell</td>
</tr>
</table>

colspan and rowspan

Colspan and rowspan are scaling attributes used with table cells which scale a table cell with respect to other cells.

By default, a table cell takes a space of one colomn and one row, which can be considered a unit.

Both colspan and rowspan take a positive integer which decides how much space a cell will take in the direct in the direction of column or row.

For example if the value of colspan is 3, then the cell will take space equal to 3 cells and it will cover 3 columns.

<table border='1px'>
<tr><td colspan='3'>cell</td><td>cell</td><td>cell</td></tr>
<tr><td rowspan='2'>cell</td><td>cell</td><td>cell</td><td colspan='2'>cell</td></tr>
<tr><td>cell</td><td>cell</td><td>cell</td><td>cell</td></tr>
<tr><td colspan='5'>cell</td></tr>
</table>
