An image can be used just for the display as well as a link if it is inside an anchor tag.

But sometimes, we need an image to contain multiple links so that it will redirect us to diffent websites when clicked on its different parts. This can be achieved by Image Mapping.

For image mapping, <map> tag is used.
An image is added using img tag and given a usemap attribute. Its value is a string after a # sign.

<img src="Images/MapExample.JPG" usemap="#map">

map tag is used with a name attribute which has the same value as the usemap without the # sign.

To mark the area to click we use an area tag. The area is marked by shape and coords attributes. The coordinates for the point are given from left and top in pixels. We can use following markings,

Circle:It can mark circular areas. In coords, an X, a Y coordinate and a radius is given.

<area shape="circle" coords="270,190,125" href="Flower.html">

Rect:It can mark square and rectangular areas. In coords, its top-left and bottom-right corner coordinates are given.

<area shape="square" coords="50,350,450,350" onclick="alert('Thankyou')">

Poly:It can be used to mark irregular shapes. The X&Y coordinates of each point of the area is entered as value.

<area shape="poly" coords="321,6,46,356,68,68,,342,65,8,523,3,523" href="Index.html">

Default:It covers the entire picture.

<area shape="default">

The image map can also trigger javascript events. Try to click on the text on the image.
