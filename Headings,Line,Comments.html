HTML includes six levels of headings, which are ranked according to importance.

These are <h1>,<h2>,<h3>,<h4>,<h5> and <h6>

Horizontal Lines

To create horizontal lines, use the <hr> tag.

Comments

The browser does not display comments, but they help document the HTML and add discriptions, reminders and other notes.

As you can see, the comment is not displayed in the browser.
