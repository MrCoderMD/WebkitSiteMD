HTML uses some elements to show something on a web page. These elements are called tags. There are mainly two types of tags in HTML:

Container tags: As the name suggests, thses tags contain some data inside themselves. These come in pairs so they are also called double tags.
EX.html tags(<html>...<\html>),paragraph tags(<p>...<\p>)

Empty tags: These tags come with a single tag. That's why they are called single tags also. They don't contain any data but take them only as their attributes.
EX.br tag(<br>), img tag(<img>)

Tags also use attributes which help the programmer to input further data into the tags. Some of these attributes are common for every tag like bgcolor and color, while some attributes are exclusive for a specific tag like alt in img tag.
Some tags are optional like align for paragraph and heading tags while some attributes are mandatory in order to make them work like src in img tag.



We will learn all about these tags and attributes in upcoming chapters.
